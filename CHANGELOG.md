# Change Log

All notable changes to the "ahoy-lang" extension will be documented in this file.

## [0.1.0] - 2024-10-28

### Added
- Initial release
- Syntax highlighting for Ahoy language
  - Keywords (if, then, else, loop, return, etc.)
  - Constants with :: operator
  - Functions with :: and | syntax
  - Operators (plus, minus, times, div, etc.)
  - Word operators (is, greater_than, less_than, etc.)
  - Comments with ? prefix
  - Strings, characters, and numbers
  - Types (int, float, string, bool, etc.)
- Language configuration
  - Auto-closing pairs for brackets
  - Comment toggling
  - Bracket matching
- File association for .ahoy files

### Coming Soon
- Language Server Protocol (LSP) integration
- Code completion
- Error diagnostics
- Go to definition
- Hover information
